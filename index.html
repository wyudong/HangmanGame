HangmanGame

Hi! Welcome to this page for introducing and archiving my Strikingly interview project - a word-guessing game on iOS.

Walkthrough

My project is base on the MVC design pattern.

Model

The model RESTfulAPIManager interacts with the HTTP connection and implements all the given APIs in 5 Objective-C methods with completion handlers. Another model HMString is a subclass of NSString, which reads each character of a word and replace it with other characters. And HMGameMangaer has jut one shared instance telling the program if player wants to start a new game or continue the last saved game.
